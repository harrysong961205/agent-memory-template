# 만나의 항아리 — Jar of Manna

> 나에게 보내는 생각을 담는 항아리. 실행할 일이 아니라, 머릿속 어딘가에 묵혀 두고
> 싶은 성찰·아이디어·"언젠가의 나에게" 보내는 메모.
>
> 출애굽기 16장 — 광야의 만나는 대부분 다음 날 아침이면 상했다. 단 한 오멜만,
> 항아리에 담겨 증거궤 앞에 보관됐다. "너희 대대로 간수하라." 지나가는 생각은
> 그날의 만나처럼 흘려보내도 된다. 간직할 가치가 있는 하나를 이 항아리에 담는다.

## 쓰는 법

- **추가 전용**: 최신 항목을 맨 위에. 지우지 않는다.
- **할 일이 아니다**: 여기 담긴 글은 태스크가 아니다. 에이전트는 이 항목을 멋대로
  `ROADMAP.md` / `ACTIVE_WORK.md`로 옮기거나 실행하지 않는다. 때가 오면 사용자가
  직접 "승격"시킨다. 그 전까지는 그냥 항아리 안에서 쉰다.
- **담는 말**: "만나의 항아리에 넣어줘", "만나에 넣어", "이건 만나행" — 또는
  실행할 일이 아닌 생각을 "기억해둬"라고 할 때.
- **형식**:
  ```
  ## YYYY-MM-DD — {제목}

  {생각 — 사용자의 목소리 그대로}

  **왜 보관**: {한 줄 — 왜 간직하는가}
  ```
