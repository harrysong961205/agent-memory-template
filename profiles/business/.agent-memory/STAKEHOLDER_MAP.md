## 변경 시 체크 규칙

### 외부에 나가는 메시지/자산을 바꿀 때
1. `BRAND_CONTEXT.md` 보이스와 일치하는지 확인
2. 이 맵에서 그 자산을 쓰는 **모든 기능** 확인
3. 각 기능의 해당 문서 동기화

### 가격·패키지·정책을 바꿀 때
1. 영향받는 모든 기능 식별 (SHARED 항목 참조)
2. `DECISION_LOG.md`에 결정 기록
3. 각 기능 문서 + 외부 자산 업데이트

### 새 활동/이니셔티브를 추가할 때
1. SHARED인지 특정 기능인지 분류
2. 이 파일에 새 항목 추가
3. 관련 `FUNC_*.md`에 반영
