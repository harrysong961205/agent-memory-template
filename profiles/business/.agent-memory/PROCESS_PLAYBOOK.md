# Process Playbook

> 반복되는 표준 업무 절차(SOP)를 모은다. "이거 어떻게 하더라"가 나오면 여기에 적는다.
> 일회성 결정은 `DECISION_LOG.md`, 반복 절차는 여기.

> 프로세스 형식:
>
> ## {프로세스 이름}
> - **기능**: BD / SL / MK / OP / HR / FN
> - **트리거**: 언제 시작되는가
> - **단계**: 1) … 2) … 3) …
> - **소유자**: 책임자
> - **산출물**: 끝나면 무엇이 남는가
> - **연결 문서**: 관련 `FUNC_*.md` / 템플릿

---
